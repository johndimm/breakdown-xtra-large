Breakdown
getting the feel of data
  • Olympics demo: A quick way to try this interface without downloading or installing anything.
  • with Mint or Personal Capital: sign up for either service, then install the Google Chrome Extension. Slice and dice your financial data by account, category, description, and time.
  • use your own data: click import to load any spreadsheet. Note that your data is NOT uploaded, only transferred to local storage in your browser. The best spreadsheets for use in Breakdown contain some columns that have a limited number of distinct values. Things like city, month, age in years, category, payee. A good example is the data for the Olympics demo in a google spreadsheet.
